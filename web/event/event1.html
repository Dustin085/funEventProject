活動介紹

孩子們初步探索互動式網頁製作的最佳首選，一日體驗網頁製作的全部流程，對美感、程式設計有興趣的朋友們趕快來吧！

孩子將親手打造不同的程式專案，得到程式開發的實務經驗、解決各種複雜問題的能力。

將獲得專屬的程式作品集、累積個人專業程式認證。

師資介紹

由"網站設計工作室-有趣活動"的資深成員們擔任講師，有趣活動工作室自成立以來便積極投入在親子活動體驗上，這次帶來的課程是成員們最最擅長的網站設計，誓要帶給孩子們最優質最清楚的體驗課程。

活動流程

09:00~11:00 => ux課程，初步了解使用者體驗的含意，進行使用者體驗分析

11:10~12:00 => ui課程，操作目前業界最流行的figma進行ui介面設計

12:00~13:00 => 午餐時間

13:00~14:00 => ui課程延續，利用figma了解網頁的基本架構

14:10~17:00 => 學會使用html製作網頁主體並且使用css對網頁進行裝飾，最後使用JavaScript撰寫網頁的互動效果
